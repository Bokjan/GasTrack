# GasTrack 需求分析

## 1. 项目背景与目标

GasTrack 是一款面向全球用户的油耗记录与分析系统：
- 记录每次加油的详细信息（油量、费用、里程等）
- 追踪车辆油耗趋势，辅助用车决策
- 多车辆管理，满足家庭或车队需求
- 支持不同国家/地区的计量单位、货币和语言

## 2. 用户角色

| 角色 | 描述 |
|------|------|
| 普通用户 | 注册登录后记录加油数据、查看报表 |
| 家庭管理员 | 创建家庭/群组，邀请成员，查看群组内车辆数据 |
| 系统管理员 | 后台管理、用户管理、数据统计 |

## 3. 核心功能模块

### 3.1 用户与认证 (P0)
- 邮箱注册/登录
- 第三方登录（Google / Apple / 微信）
- 个人资料：头像、昵称、时区、偏好语言、默认币种和计量单位
- 修改密码、忘记密码（邮件重置）
- 账号注销（GDPR 合规数据删除）(P1)

### 3.2 车辆管理 (P0)
- 添加车辆：品牌、型号、年份、油箱容量、燃油类型（汽油/柴油/混动）
- 编辑/删除车辆
- 多车辆切换（支持默认车辆设置）
- 车辆照片上传 (P1)
- 车辆里程校准 (P2)

### 3.3 加油记录 (P0 - 核心功能)
- 记录加油信息：日期时间、加油站、油量、单价、总费用、当前里程
- 是否加满标记（影响油耗计算准确性）
- 支持手动输入和拍照识别加油小票 (P2)
- 加油记录的编辑/删除
- 记录列表：按时间排序，支持筛选（按车辆/日期范围/加油站）

### 3.4 油耗统计与报表 (P0)
- 单次油耗计算：L/100km 或 MPG（根据用户偏好）
- 油耗趋势图：折线图展示油耗变化
- 费用统计：按月/季度/年的加油费用汇总
- 费用趋势图：柱状图展示费用变化
- 多车辆对比 (P1)
- 数据导出 CSV/PDF (P1)

### 3.5 多语言支持 (P0)
- 第一期：中文简体、英文
- 第二期：日语、韩语、繁体中文、西班牙语、德语、法语
- 语言自动检测（浏览器/系统语言）+ 手动切换
- 后端错误消息也需国际化

### 3.6 多国家/地区与多币种 (P0)
- 支持主要国家/地区设置
- 计量单位自动关联：
  - 公制：升(L)、公里(km)、L/100km
  - 英制：加仑(gal)、英里(mi)、MPG
- 多币种支持：CNY、USD、EUR、JPY、GBP 等
- 汇率参考（只读展示，不做实时兑换）(P2)

### 3.7 家庭/群组管理 (P1)
- 创建家庭群组
- 邀请码/链接邀请成员
- 群组内车辆数据汇总查看
- 成员权限管理（管理员/成员）

### 3.8 通知与提醒 (P2)
- 保养提醒（按里程/时间）
- 定期加油提醒
- 异常油耗预警

## 4. 非功能性需求

### 4.1 性能
- 页面首屏加载 < 2s
- API 响应时间 < 200ms（P95）
- 支持 1 万并发用户

### 4.2 安全
- HTTPS 全链路加密
- JWT Token + Refresh Token 认证
- 密码 bcrypt 加密存储
- SQL 注入/XSS 防护
- 接口限流（Rate Limiting）

### 4.3 可用性
- 响应式设计（PC + 移动端）
- PWA 支持，可离线使用基本功能 (P1)
- 无障碍访问（WCAG 2.1 AA）(P2)

### 4.4 数据合规
- GDPR 合规（数据导出、删除权）
- 用户数据加密存储
- 隐私政策与用户协议

## 5. 优先级说明

| 优先级 | 含义 | 时间范围 |
|--------|------|----------|
| P0 | 第一版必须实现 | 第一期（8-10周） |
| P1 | 重要但可延后 | 第二期（4-6周） |
| P2 | 锦上添花 | 第三期及以后 |
